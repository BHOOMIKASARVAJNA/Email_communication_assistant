# Email Communication Assistant

## Description
An AI-powered assistant that generates professional or informal emails
based on user-provided context and tone.

## Inputs
- Context
- Tone

## Outputs
- Draft email
- Alternative version
- Subject line suggestions

## Technologies Used
- Python
- Groq API
- NLP Prompt Engineering

## How to Run
1. Install dependencies
2. Set API key in .env
3. Run app.py
